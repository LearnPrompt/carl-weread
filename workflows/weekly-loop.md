# Workflow: weekly-loop

## 触发

用户说：

- 本周阅读复盘
- 看看哪些阅读真的进入工作了
- 下周该保留哪条阅读主线

## 目标

这不是年度报告，也不是数据展示。目标是判断：哪些阅读真的进入行动，哪些只是逃避式收藏。

## 输入

- 本周微信读书阅读统计
- 本周新增笔记/划线
- 本周 Obsidian 日记和项目进展
- 本周内容选题和输出

## 输出

```markdown
# 本周阅读行动闭环

## 本周真正读进去的东西
...

## 已经进入工作的证据
...

## 逃避式收藏
...

## 下周只保留一条阅读主线
...

## 下周 3 个 Today Chapter 候选
1. ...
2. ...
3. ...
```

## 规则

- 不用「读了多少」制造成就幻觉。
- 必须找阅读和真实项目之间的连接证据。
- 下周主线只能有一条。
- 如果本周读得很散，要直接指出，不美化。
